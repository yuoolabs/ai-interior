# 真实页面动作规则

## 已确认可用的动作
1. 列表页点击 `新建页面` 可以进入编辑器
2. 编辑器左侧点击 `标题文字` 会立刻把组件加到中间画布
3. 右侧点 `页面设置` 可以切回页面级配置
4. 页面名称输入框可以通过占位文案 `请设置微页面名称` 定位
5. 保存按钮文本是 `保 存`
6. 预览按钮文本是 `预 览`

## 执行顺序要求
1. 进入编辑器后先填页面名称
2. 再开始按顺序添加组件
3. 每加一个组件后检查中间画布是否出现对应组件
4. 如果右侧已经切到组件配置，不要直接继续找页面名称，要先点回 `页面设置`

## 当前保存行为结论
1. 即使 `图文广告` 没图片、`优惠券` 没券、`商品` 没商品，也可以先保存草稿
2. 保存成功后会生成新的页面记录
3. 所以后续执行策略应改为：
   先允许保存草稿
   再把空模块作为“待补内容”回传
