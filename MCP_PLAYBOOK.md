# Chrome DevTools MCP 执行说明

## 目标
把“页面方案”转换成浏览器里的稳定动作，避免一边思考一边点页面。

## 执行顺序
1. 打开 `https://smp.iyouke.com/dtmall/designPage`
2. 点击 `新建页面`
3. 填写页面名称
4. 从左侧组件面板按顺序添加组件
5. 每添加一个组件后检查右侧是否出现配置区
6. 全部完成后点击 `保存`
7. 再点击 `预览`

## 停机规则
1. 页面跳转失败时停止
2. 组件面板找不到目标组件时停止
3. 组件数量超限制时停止并降级
4. 保存按钮不可点时停止

## 输出要求
1. 成功创建的页面名称
2. 成功添加的组件列表
3. 替代实现列表
4. 未完成原因
